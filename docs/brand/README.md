# Pool Conductor Brand Identity

This is the working source for the Pool Conductor brand. It lives in the
marketing repository but outside Astro's `src` and `public` directories, so
nothing in this folder is included in the public website build.

## Documents

- [Brand foundation](./brand-foundation.md): purpose, position, promise, and principles
- [Audiences](./audiences.md): homeowners and pool professionals
- [Voice and messaging](./voice-and-messaging.md): how Pool Conductor should sound
- [Visual identity](./visual-identity.md): color, type, layout, photography, and illustration
- [Logo identity](./logo-brief.md): the selected Open Channel monogram and usage rules

## Working Rule

Pool Conductor should make pool ownership feel quieter. Every design and message
should reduce perceived work, make the product feel dependable, and respect the
knowledge of the person installing it.
